Midway

Full stack programming for the future

Quick Guide →

Simplicity First

Using IOC to manage application dependencies helps maximize the maintainability and scalability of the application in web development, and developers do not need to focus on Object creation.

Future-oriented

Enjoy the development experience of typescript, by decorator and dependency injection, make application development smooth and natural, and focus on encoding.

Embrace community

The Koa middleware and various eggjs plug-ins make it easy for users to get started and migrate quickly, and reduce development and deployment costs.

# install
npm install -g midway-init
midway-init
npm install
npm run dev

COMPATIBILITY NOTE

midway requires Node.js >= 8.
