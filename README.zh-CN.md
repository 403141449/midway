## 安装

```bash
$ npm install midway --save
```
需要 Node.js >= 8.0.0。

## 特性

- ✔︎ 精致的中间件分组架构和兼容性
- ✔︎ 可扩展的插件体系生态
- ✔︎ 良好的应用分层和解耦能力
- ✔︎ 面向未来的开发体验

## 快速开始

使用以下命令

```bash
$ npm install midway-init -g
$ midway-init
$ npm install
$ npm run dev
$ open http://localhost:7001
```
