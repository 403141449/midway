Midway

面向未来的全栈应用开发框架

快速上手 →

简洁至上

使用 IoC 将应用依赖管理起来，最大化的帮助应用在 web 开发中提升可维护性和扩展性，开发者无需关注对象创建。

面向未来

享受 Typescript 的开发体验，通过装饰器和依赖注入，让应用开发变的流畅自然，可以专注业务逻辑编码。

拥抱社区

兼容社区 koa 中间件以及各种 Eggjs 插件，方便用户快速上手和迁移，降低开发和部署成本。

# 安装
npm install -g midway-init
midway-init
npm install
npm run dev

注意

请确保你的 Node.js 版本 >= 8。
